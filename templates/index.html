Docker + Flask + Render + Nuclei

Nuclei Security Scanner

A cloud-based vulnerability scanner that runs Nuclei inside a Docker container and displays JSONL findings in a web dashboard.

Start a Scan

Enter an authorized test website URL. For this assignment, use demo sites such as https://demoqa.com/ or http://testphp.vulnweb.com.

System Flow

Educational use only. Only scan authorized targets, test websites, or your own applications.
1. User enters URL
2. Flask receives request
3. Nuclei scans target
4. JSONL results are stored
5. Dashboard displays findings
