Docker + Flask + Render + Nuclei

Nuclei Security Scanner

A cloud-based vulnerability scanner that runs Nuclei inside Docker, stores scan output in JSONL format, and presents findings through a polished security dashboard.

System Flow

The Flask API receives the target URL, runs Nuclei inside the Docker container, stores the JSONL results, and sends the findings back to the dashboard.

1 User enters URL
2 Flask receives request
3 Nuclei scans target
4 JSONL results are stored
5 Dashboard displays findings
